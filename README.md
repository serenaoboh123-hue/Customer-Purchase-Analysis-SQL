# Customer Purchase Analysis Using SQL

## Project Overview
This project analyzes customer profiles, product information, and purchase history using SQL to answer business questions about customer behavior and product sales.

## Objectives
- Analyze customer purchase behavior.
- Explore product categories and pricing.
- Practice SQL joins and aggregate functions.
- Generate business insights from relational data.

## Dataset
The project uses three related datasets:
- Customer Profile
- Products
- Purchase History

## SQL Skills Demonstrated
- INNER JOIN
- WHERE filtering
- Aggregate Functions (AVG)
- Table Aliases
- Multi-table Queries

## Sample Business Questions
- Which customers purchased products in the Dairy category?
- What is the average price of products purchased by female customers?
- Which products are most frequently purchased?
